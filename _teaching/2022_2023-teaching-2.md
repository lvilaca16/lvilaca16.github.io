## Course Description

### Goals

Students should develop skills in the area of multimedia communication systems, namely on multimedia digital signal processing, system architectures, protocols and applications. At the end of the course, students should
have the following general competences:

- Understand the concepts of multimedia acquisition and digital representation
- Understand the techniques for multimedia signal processing
- Understand and be able to analyse the impact of some operations on audiovisual content
- Know the protocols used to convey multimedia content
- Specify and develop modules of a multimedia communication system

### Program

1. Introduction to audiovisual content processing and transmission
- Basic concepts
- Multimedia Signal Representation
- Basic operations on images

2. Coding techniques
- Entropy coding
- Transform coding
- Compression
- International standards: JPEG, MPEG, H264 / AVC, H265 / HEVC;

3. Multimedia transmission
- Multiplexing and Transporting Multimedia Signals
- Digital Television and Radio Broadcast: DVB; DAB;
- Protocols for Multimedia Communications: RTP, RTSP, MPEG-DASH, HLS

4. Machine Learning and Deep Learning for Multimedia
- Learning approaches
- Data and learning pipeline
- Frameworks and models
